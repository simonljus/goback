# goback
Go API-service 

uses [Gin](https://github.com/gin-gonic/gin)
## Authentication
* [gin-contrib/sessions](https://github.com/gin-contrib/sessions)

* [following auth example](https://github.com/Depado/gin-auth-example/blob/master/main.go)

# How to run
* install go
* install packages
* go run api.go
  
## Features
* Add message
* Edit message
* Delete message
* Get all messages
* Authentication
